⚡ Quick Start

Get Started in 3 Steps

Install Robot Framework and write your first Test Case in just a few minutes.

📦 Step 1 — Install
$pip install robotframework
$robot --version
# Check the installed version
📝 Step 2 — Create a .robot file
*** Settings ***
Library Collections
*** Test Cases ***
My First Test
    Log Hello Robot!
▶️ Step 3 — Run Tests
$ robot my_test.robot
# Run a single file
$ robot tests/
# Run an entire folder
$ robot -d results tests/
# Save reports to results/

🖥️ UI Testing
🌐
SeleniumLibrary
UI Testing
Test Web UI via Selenium WebDriver. Supports Chrome, Firefox, Edge — click, fill forms, verify elements, take screenshots, and more.
$ pip install robotframework-seleniumlibrary
📖 Docs
🎭
Browser Library
UI Testing
Powered by Playwright — test modern web apps faster than Selenium. Supports async, network interception, and has built-in auto-waiting.
$ pip install robotframework-browser
📖 Docs
🔌 API Testing
🔗
RequestsLibrary
API Testing
Test REST APIs by sending HTTP requests (GET, POST, PUT, DELETE). Manage sessions, auth, headers, body, and verify responses.
$ pip install robotframework-requests
📖 Docs
📋
JSONLibrary
API Testing
Manage JSON data: parse, validate, compare, and search keys via JSONPath. Pairs perfectly with RequestsLibrary for REST API testing.
$ pip install robotframework-jsonlibrary
📖 Docs
🗄️ Database Testing
🗄️
DatabaseLibrary
Database
Connect to and test databases. Run SQL queries, verify data — supports MySQL, PostgreSQL, Oracle, SQLite, and others via Python DB-API.
$ pip install robotframework-databaselibrary
📖 Docs
📱 Mobile Testing
📱
AppiumLibrary
Mobile
Test mobile apps on iOS and Android via Appium. Supports Native Apps, Hybrid Apps, and Mobile Web. Requires an Appium Server connection.
$ pip install robotframework-appiumlibrary
