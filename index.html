⚡ Quick Start

เริ่มต้นใน 3 ขั้นตอน

ติดตั้ง Robot Framework และเขียน Test Case แรกของคุณได้ภายในไม่กี่นาที

📦 Step 1 — ติดตั้ง
$pip install robotframework
$robot --version
# ตรวจสอบ version ที่ติดตั้ง
📝 Step 2 — สร้างไฟล์ .robot
*** Settings ***
Library Collections
*** Test Cases ***
My First Test
Log Hello Robot!
▶️ Step 3 — รัน Test
$ robot my_test.robot
# รันไฟล์เดียว
$ robot tests/
# รันทั้ง folder
$ robot -d results tests/
# บันทึก report ไว้ที่ results/

📚 Libraries

Library ทั้งหมด

เลือก Library ตามประเภทการทดสอบที่ต้องการ ครอบคลุมทุก use case

📦 Standard Library
รวมมากับ robotframework
🗂️
Collections
Standard
จัดการ List, Dictionary, Set และ data structure ต่าง ๆ มี keyword ครบสำหรับ append, remove, sort, compare รวมถึง nested structures
✓ built-inpip install robotframework

DateTime
Standard

✓ built-inpip install robotframework

SeleniumLibrary
UI Testing
ทดสอบ Web UI ผ่าน Selenium WebDriver รองรับ Chrome, Firefox, Edge คลิก, กรอกข้อมูล, ตรวจสอบ element, screenshot ฯลฯ
$ pip install robotframework-seleniumlibrary
📖 Docs
🎭
Browser Library
UI Testing
ใช้ Playwright เป็น engine ทดสอบ Modern Web ได้เร็วกว่า Selenium รองรับ async, network interception, มี auto-waiting ในตัว
$ pip install robotframework-browser

🔌 API Testing
🔗
RequestsLibrary
API Testing
ทดสอบ REST API ส่ง HTTP request (GET, POST, PUT, DELETE) จัดการ session, auth, header, body และ verify response
$ pip install robotframework-requests
📖 Docs
📋
JSONLibrary
API Testing
จัดการ JSON data: parse, validate, เปรียบเทียบ, ค้นหา key ด้วย JSONPath ใช้คู่กับ RequestsLibrary สำหรับ REST API testing
$ pip install robotframework-jsonlibrary

DatabaseLibrary
Database

$ pip install robotframework-databaselibrary
